Capstone Project

Plotting and Navigating a Virtual 2D Maze
______________________________________________________________________
Install 
This project requires Python 2.7 with the following library installed:

numpy
networkx(https://pypi.python.org/pypi/networkx/1.7)
turtle
_______________________________________________________________________
Run 
In a terminal window , run the following command:

python tester.py test_maze_**.text

At the end click the turtle graphics window to close the run.
